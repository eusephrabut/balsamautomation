# Page Object model Test automation framework using Selenium with Java, TestNG and Maven-
This is for Technical Assessment for Balzam Brands

TestNG is used as test framework.

Dependency
Java
Maven

###libraries used
Selenium
TestNG
log4j
Extent Reports

### Having trouble with my GitHub account. Run the automation Locally
1. Extract and move folder to C:\

### Steps execute the tests
```
1.Navigate to src/test/java/testcase
2. Right click on "BalsamTest.java" then Run As TestNG Test
```
